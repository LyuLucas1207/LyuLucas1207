# Lsr-Web 部署说明
# Site: https://www.lucaslyu.com（可选镜像域 www.lyulucas.com，见 NFX-Edge）
# 生产 API/WS: 见 .env.prod（例：https://api.lucaslyu.com）
# 本地网关: 见 .env.dev / Lsr-Backend TRAEFIK_HTTP_PORT

# 部署前检查清单：
# 1. public 下已有 favicon.svg、icons.svg、shaders/（构建会复制到 dist）
# 2. Bing：替换 BingSiteAuth.xml 中的验证码后，部署访问 https://www.lucaslyu.com/BingSiteAuth.xml
# 3. Google Search Console：按控制台要求上传 html 验证文件到 public/（文件名以控制台为准）
# 4. 确认 .env.prod / .env.prod.local 中 VITE_API_URL、VITE_WS_URL
# 5. 生产镜像构建：./start.sh 或 docker build（Dockerfile 使用 build:prod-local）
# 6. SPA：Nginx 已配置 try_files … /index.html；Netlify / Cloudflare Pages 可用 public/_redirects
